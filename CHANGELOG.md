# Changelog

## [Unreleased]
- More options
- Add record editing
- Credits

## [v1.1-alpha] - 2026-01-30
- Mouse actions are now recorded together with mouse movements, instead of keyboard inputs.
- When a mouse action is recorded, it's position is also recorded.
- Added save feature
- Added load feature
- 2026-01-29 changelog now considered [v1.0-alpha], instead of [v0.3-alpha]

## [v1.0-alpha] - 2026-01-29
- Reworked the look of the GUI
- Refactored the GUI

## [v0.2-alpha] - 2026-01-28
- Refactored the code, to improve readability
- Fixed a bug, that let you record while playing a record. Now play and record check the state before doing anything.

## [v0.1-alpha] - 2026-01-26
- Initial alpha release
- Record and playback keyboard input
- Record and playback mouse buttons, scroll wheel, and position
- Basic GUI for hotkeys and mouse mode selection
- Configurable hotkeys for recording and play
