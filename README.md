# AutoHotkey(AHK) Macro Recorder

A GUI-based AutoHotkey v2 script to record and replay keyboard and mouse macros.

## Features
- Record keyboard and mouse input
- Replay macros with precise timing
- GUI controls for start/stop/play
- Configurable hotkeys and mouse position mode

## Requirements
- AutoHotkey v2

## Usage
1. Run `src/MacroRecorder.ahk`.
2. Use the GUI or hotkeys to record and play macros.

## Hotkeys
- F1 - Start/Stop recording
- F2 - Play recording
- Alt+D - Exit
- Alt+U - Reload script
